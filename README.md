## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/shoe-configuration.git
```

Go to the project directory

```bash
  cd shoe-configuration
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
